# Implementation Logs

This directory contains chronological, evidence-backed records of material
engineering work performed with Claude Code.

Each file represents one task boundary, not one edit or command. Logs may cover
implementation, audits, debugging, validation, documentation, maintenance,
research, or operational work. Repository state and authoritative project
contracts remain the source of truth; these files are concise execution records.

File names are UTC-sortable:

```text
YYYY-MM-DDTHHMMSSffffffZ--task-slug.md
```

Do not place credentials, secret values, raw environment dumps, or unnecessarily
large logs in this directory.
